HomeLab Playground

Browser-based Utilities
for Homelab Enthusiasts

A collection of lightweight, client-side tools designed to make your infrastructure management easier. No tracking, no servers, just code.

MAC Formatter

Convert between Cisco, Colon, and Dash formats instantly. Validates length and strips invalid characters.

Readme Linker

Generate perfectly formatted Markdown links for your documentation. Auto-extracts titles from URLs.

Cron Tool

Parse complex cron schedules into human-readable text and generate standard schedules with a GUI.

Mermaid Editor

Live preview editor for Mermaid.js diagrams. Supports Flowcharts, Sequence diagrams, and Gantt charts.
